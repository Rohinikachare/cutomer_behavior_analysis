# customer_behavior_analysis

## Overview

This project demonstrates an end-to-end data analytics workflow using Python, SQL databases, and Power BI. The objective of the project is to extract insights from raw data by performing data cleaning, exploratory data analysis (EDA), SQL querying, visualization, and reporting.

The project covers:

* Data loading and preprocessing in Python
* Exploratory Data Analysis (EDA)
* Data cleaning and transformation
* SQL analysis using PostgreSQL, MySQL, and SQL Server
* Interactive dashboard creation in Power BI
* Report preparation and presentation design
